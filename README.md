# Review-of-Reviews (CSCI 4405)
**Team:** Lucas Montoya & Luis Escamilla

### Overview
An AI-driven tool to summarize product reviews using logic-based topic modeling and heuristic search.

### Current Progress (Presentation I)
- Dataset acquired (Kaggle E-Commerce Reviews).
- Logic-based topic inference engine implemented.
- Transparency-focused trust scoring module (Ethics mitigation).

### How to Run
1. Place `Womens Clothing E-Commerce Reviews.csv` in the `/data` folder.
2. Run `python src/summarizer.py` to see the logic engine process a sample of reviews.

### Dataset:
https://www.kaggle.com/datasets/nicapotato/womens-ecommerce-clothing-reviews
